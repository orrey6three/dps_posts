## Как добавить свой пост

### Через админку (рекомендуется)

1. Откройте `/admin`
2. Войдите с паролем
3. Нажмите "+ Добавить пост"
4. Заполните форму:
   - **Название**: краткое описание места
   - **Адрес**: улица или ориентир (опционально)
   - **Широта**: координата (например, 55.2317)
   - **Долгота**: координата (например, 63.2892)
5. Нажмите "Сохранить"

### Через SQL (для массового добавления)

```sql
INSERT INTO posts (title, address, latitude, longitude) VALUES
('Название поста', 'Адрес (опционально)', 55.2317, 63.2892);
```

## Как найти координаты

### Способ 1: Google Maps

1. Откройте [Google Maps](https://maps.google.com)
2. Найдите нужное место в Шумихе
3. Кликните правой кнопкой на место
4. Выберите первую строку (это и есть координаты)
5. Формат: `55.231700, 63.289200`
   - Первое число = Широта (latitude)
   - Второе число = Долгота (longitude)

### Способ 2: Яндекс.Карты

1. Откройте [Яндекс.Карты](https://yandex.ru/maps)
2. Найдите место
3. Кликните правой кнопкой
4. Выберите "Что здесь?"
5. Координаты отобразятся внизу

### Способ 3: OpenStreetMap

1. Откройте [OpenStreetMap](https://www.openstreetmap.org)
2. Найдите место
3. Кликните правой кнопкой
4. Выберите "Show address"
5. URL изменится и будет содержать координаты

## Рекомендации по созданию постов

### Название

✅ Хорошие названия:
- "Пост ДПС - Центр города"
- "Пост ДПС - Трасса М51 въезд"
- "Пост ДПС - Район автовокзала"

❌ Плохие названия:
- "Пост 1"
- "ДПС"
- "Там где всегда стоят"

### Адрес

- Укажите улицу или известный ориентир
- Примеры: "ул. Ленина", "Район школы №1", "Трасса М51"
- Можно оставить пустым, если адрес неизвестен

### Координаты

- Точность: 4-6 знаков после запятой достаточно
- Для Шумихи:
  - Широта примерно: **55.20 - 55.25**
  - Долгота примерно: **63.25 - 63.35**
- Проверьте на карте перед добавлением!

## Массовое добавление

Если у вас есть список постов в Excel/Google Sheets:

1. Подготовьте данные в формате:
   ```
   Название | Адрес | Широта | Долгота
   ```

2. Сгенерируйте SQL запрос:
   ```sql
   INSERT INTO posts (title, address, latitude, longitude) VALUES
   ('Пост 1', 'Адрес 1', 55.2317, 63.2892),
   ('Пост 2', 'Адрес 2', 55.2320, 63.2900),
   ('Пост 3', 'Адрес 3', 55.2310, 63.2880);
   ```

3. Выполните в Supabase SQL Editor

## Удаление постов

### Через админку

1. Откройте `/admin`
2. Найдите пост
3. Нажмите на иконку корзины
4. Подтвердите удаление

### Через SQL

```sql
-- Удалить конкретный пост по названию
DELETE FROM posts WHERE title = 'Название поста';

-- Удалить все посты (ОСТОРОЖНО!)
DELETE FROM posts;

-- Удалить посты без адреса
DELETE FROM posts WHERE address IS NULL;
```

## Экспорт/Импорт

### Экспорт всех постов

```sql
-- Скопируйте результат и сохраните в CSV
SELECT 
    title, 
    address, 
    latitude, 
    longitude 
FROM posts 
ORDER BY title;
```

### Импорт из CSV

Используйте Supabase Table Editor:
1. Откройте таблицу `posts`
2. Нажмите "Insert" → "Import data"
3. Загрузите CSV файл
