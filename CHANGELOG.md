# Список реализованных функций (Релизная версия)

Данный документ содержит описание ключевых доработок и новых функций, реализованных для проекта DPS45.

## 1. Личный кабинет Администратора (Обновлено)
*   **Панель управления (Dashboard):** Добавлена сводная статистика по количеству меток, зарегистрированных пользователей и общего числа голосов.
*   **Управление пользователями:** Появился раздел "Пользователи", где админ может видеть список всех зарегистрированных участников, дату их регистрации и количество созданных ими меток. Реализована возможность удаления пользователей.
*   **Продвинутое создание меток:** В админ-панели теперь можно не только выбирать место на карте, но и указывать категорию, добавлять комментарий и теги при создании метки.
*   **Улучшенный интерфейс:** Полностью обновлен дизайн с использованием вкладок, современных карточек и уведомлений (toast).

## 2. Пользовательский функционал
*   **Удаление собственных меток:** Пользователи теперь могут удалять метки, которые они создали сами. Кнопка удаления появляется автоматически в карточке метки, если текущий пользователь является её автором.
*   **Смена темы (Темная/Светлая):** Добавлен переключатель тем в верхнем углу карты. Выбор пользователя сохраняется в браузере.
*   **Быстрый доступ к админке:** Для пользователей с правами администратора в углу карты отображается кнопка быстрого перехода в панель управления.

## 3. Исправления и UX улучшения
*   **Исправление удаления меток:** Исправлена ошибка «ID метки не найден», возникавшая из-за несоответствия имен полей в базе данных и фронтенде. Удаление теперь работает корректно.
*   **Закрытие окон по ESC:** Все модальные окна и всплывающие панели теперь закрываются нажатием клавиши Escape.
*   **Исправление точности нажатия:** Увеличена область "хитбокса" меток на карте и исправлена проблема, когда при нажатии на метку иногда срабатывало добавление новой (прервано всплытие события).
*   **Локализация:** Уточнены названия кнопок голосования («Актуально» / «Не актуально») для лучшего понимания.
*   **Адаптивность:** Исправлено "проваливание" панелей на мобильных устройствах, добавлена поддержка безопасных зон (Safe Areas).

## Технические детали
*   **Система авторизации:** Настроен проброс cookies (credentials) для корректной проверки прав владения меткой на стороне сервера.
*   **База данных:** Оптимизированы SQL-функции для получения статистики и корректной работы системы актуальности.
